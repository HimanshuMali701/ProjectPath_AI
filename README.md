# 🚀 ProjectPath AI

> From Skills → Skill Gaps → Projects → Career Growth

ProjectPath AI is an AI-powered career project advisor that helps users discover the **right projects to build** based on their skills, skill gaps, and career goals.

## 🔍 Features

- Upload your resume (PDF) or enter skills manually  
- Extract skills using NLP-based parsing  
- Expand related skills via a skill graph  
- Detect user level (Beginner / Intermediate / Advanced)  
- Identify skill gaps & missing high-demand skills  
- Recommend Top 3 personalized projects with match scores  
- Explain why each project is recommended  
- Provide datasets & starter project ZIPs  
- Show role-specific trending market skills  
- Generate a resume strength score  
- Visualize insights with interactive dashboards

## ⚙️ How It Works

1. User uploads resume or enters skills  
2. System extracts and analyzes skill set  
3. Expands skills using a predefined knowledge base  
4. Identifies skill gaps for selected roles  
5. Matches user profile with suitable projects  
6. Provides recommendations with datasets & starter code

## 📦 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/HimanshuMali701/ProjectPath_AI.git
cd ProjectPath_AI
```
### 2. Backend Setup
```
cd backend
pip install -r requirements.txt
python app.py
```
### 3. Frontend Setup
```
cd frontend
npm install
npm start
```
